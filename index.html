Finally there are some quick tips about this theme from our friends Alan Ezust and Paul Ezust:

  1. The keyword operator is used in C++ to define a new meaning for an operator symbol,such as +, -, =, *, and &.
  2. Overloaded operators provide a more compact syntax for calling functions, leading to more readable code (assuming the operators are used in ways that are commonly understood).
  3. it is possible to overload operators as member functions, or as global functions. The primary difference that you notice first is how they can be called. In particular, a member function operator requires an object as the left operand. A global function, in contrast,permits the same kinds of type conversions for either operand.
  4. Here is a way to remember which operators can be overloaded. If the symbol has a dot (.) in it anywhere, overloading is probably not allowed.
  5. Overloading the comma operator is permitted but not recommended until you are a C++ expert.
  6. It is possible to define a new meaning for a built-in operator symbol so that it can be used with operands of different types. But it is not possible to change the associativity or the precedence of a built-in operator symbol.

Qt DNA Analysis App.

By Carlos Enrique Hernandez Ibarra 01 August 2016 Guadalajara Jalisco.

[En] The DNA Analysis to the Thermo Fisher Challenge was developed with Qt/C++ Getting a population of patients,the application should determine the percentage of population that is infected. The program composed by:

  1. Download of DNA patients files (.dna format) from URL's<./li>
  2. Analysis each DNA patients file sequence in form of a string composed of a sequence of characters A,C,G and T.
  3. Calculate if the patient is infected by calculationg the number of ocurrences of all characters. If the number of ocurrences of T is greater than all the other characters, the patient is infected; if not, the patient isn't infected
  4. Determine the rounded percentage of infected population %Infected= # of Infected/Total Patients

So the first step was to download the DNA patient files. Using the QNetworkAccessManager as a object in a class called DownloadManager that inherits from QObject. We set a list of QNetworkReply to determine when there are not more files to download and stop the application. Also we used another list to contain the names of the DNA files to futher analysis.

So, in the main loop, we are going to put a DownloadManager object. The constructor of DownloadManager trigger a connection between a QNetworkAccessManager and the DownloadManager slot downloadFinished(). This function perform the analysis of DNA patient in the last setted DNA fileNote: If you aren't familiarized with this behaviour of Sinals and Slots in Qt, we recommend read this tutorial

Before annoucing the downloadFinished() method. We are going to talk about the implementation of execute that is the trigger to begin the download of DNA files. It achieve a bifurcation when the command is without parameters. The other way call the QNetworkAccessManager hosting of the download request from the url's

Finally, the downloadFinished() method. Its general purpose is to analysis all the download files counting the amount of ocurrences of the characters. Son we process each file, count all the ocurrences and calculate the percentage of infected people. We used the QFile class to achieve the file processing to save the string to a QByteArray (that contains char characters) Also we used the Patient class to hold the amount of characters A,C,G and T. And later we calculate and print the results to the terminal. ./download 1.dna 2.dna 3.dna
